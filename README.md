# Страница регистрации аккаунта

Проект в рамках обучения на курсе "Профессия Frontend-Разработчик" в онлайн-школе "Айтилогия".

## Внешний вид

![project-screen](project-image.jpg)

## Выполненные задачи:
- Вёрстка веб-страницы регистрации и авторизации по макету;
- Практика использования JavaScript для реализации необходимого функционала веб-страницы(обработка событий, изменение атрибутов html-элементов);
- Реализация обработки событий, изменения структуры DOM-элементов, простейшей валидации.

## Используемые технологии:
* HTML
* CSS
* JavaScript
